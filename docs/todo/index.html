CONFIG

Comme on partage des données entre divers scripts? Envisager un fichier de config.ini ou des variables d'environnement. Stocker ce fichier ini dans ./configs qui doit etre ignoré par git. On peut utiliser un fichier .env à la racine du projet et utiliser le module load env.
