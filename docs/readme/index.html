Sur RPI WARNING avec la Lib PILLOW

Pillow ne s'installe pas sur RPI, il faut installer les packages $ sudo apt install python3-dev libjpeg-dev zlib1g-dev Puis relancer l'installation, dans le VENV, pip install Pillow $ pip install --upgrade Pillow
